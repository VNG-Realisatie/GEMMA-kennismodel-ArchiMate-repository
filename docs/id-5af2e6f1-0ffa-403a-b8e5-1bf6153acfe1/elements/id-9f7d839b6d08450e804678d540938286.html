Omgevingswet Archi-repository ()
Thema Omgevingswet Voor 2021 staat een nieuwe Nederlandse Omgevingswet op stapel. Deze wet richt zich op onze fysieke leefomgeving. Hiermee worden onder andere bouwwerken, infrastructuur, water, bodem, lucht, landschappen, natuur en cultureel erfgoed bedoeld. Een belangrijk onderwerp, want vrijwel alle activiteiten die burgers, bedrijven en overheden uitvoeren hebben grote invloed op de fysieke leefomgeving.
